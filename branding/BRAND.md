# C35 · Crunch Capital — Brand Guide

- **Accent:** Code Crunch blue `#2463EB` (with deep `#1d4ed8`).
- **Voice:** operator-minded and rigorous. Always show the cash flows, then the model, then *why* the number is what it is. No black-box valuations — a DCF is an assumption stack you can inspect. Every model lives in SQL and Python, never a spreadsheet, so the number is reproducible and can be defended.
- **Icon motif:** a discounted cash-flow arrow meeting an upward allocation line — capital moving to its best use.
- **One-liner:** "Allocate capital like an operator — corporate finance and investing, quantified."
